# Sophia Forge Tracker

Public issue tracker for [Sophia Forge](https://www.sophiaforge.com/) — a game development app by **Sophia Foundry**.

Use [Issues](../../issues) to report bugs, request features, or ask questions.

Join our [Discord](https://discord.gg/r9W7yXk45y) community.

## How to Submit an Issue

### Bug Reports

- Describe what happened and what you expected instead
- Include steps to reproduce the issue
- Mention your OS and Sophia Forge version
- Attach screenshots or logs if possible

### Feature Requests

- Describe the problem you're trying to solve, not just the solution
- Explain your use case — how would this help your workflow?
- Check existing issues first to avoid duplicates

### General Questions

- For quick questions, try [Discord](https://discord.gg/r9W7yXk45y) first
- Search existing issues before posting — your question may already be answered

## Guidelines

- Be respectful and constructive
- One issue per report — don't combine multiple bugs or requests
- Use clear, descriptive titles
- Avoid posting sensitive information (API keys, credentials, etc.)

## Support Development

If you'd like to support Sophia Forge's development, you can contribute on [Ko-fi](https://ko-fi.com/sakushimei).
